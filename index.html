Начиная с класса шестого я занимюась самообразованием связанным с моей будущей профессией. Прошло уже много лет, я сменил место обучения, но продолжаю идти на опережение и углубляю знания, выхожу за рамки программы. За эти года я попробовал многое: ActionScript, Delphi, PHP, C++, Java, но текущим основным набором того, что я использую и знаю, считаю следующее:

  • Основной язык программирования Python;
  • Второстепенный язык - C# (изучается по программе учреждения образования);
  • Создание Web приложений с использованием фреймворка Django;
  • REST архитектура, различные HTTP запросы;
  • В качестве СУБД применяю PostgreSQL, иногда MySql;
  • Обожаю ORM, но могу строить запросы сам (знание SQL);
  • Обработка фоновых задач с использованием Celery;
  • Владею HTML. Знаю JavaScript и CSS на базовом уровне. Часто прибегаю к применению фреймворка Bootstrap;
  • Регулярные выражения;
  • JSON, XML, могу в XPath;
  • Асинхронное программирование (AsyncIO);
  • Система управления версиями (Git);
  • Развертывание проектов на сервере (BASH'a не боюсь).
